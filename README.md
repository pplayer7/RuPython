# РуПитон
Представляю вам НОВЕЙШЕЕ изобретение программирования — полное импортозамещение Питона (не Пайтона) на русском языке РуПитон!

## Идея
Моя идея заключается в том, чтобы полностью импортозаместить Питон на отечественный русский язык (я не успел к 1-ому марта 2026), чтобы наши специалисты в информационной сфере не морочили голову этим английским, который навязал нам запад! Я считаю, что наши крупные корпорации просто на-ду-ли с этими английскими языками программирования, а поэтому я рад вам представить мое решение!
## Основные замены
Данный язык полностью интуитивен, допустим, "печать" является заменой "print", а "если" — "if". Полный список засекречен, ибо ∎∎∎∎∎∎∎∎∎, но будьте уверены, что с ними все нормально! (В дробных числах не удалось заменить точки на запятые, но на это выделили ₽60млрд, придется немного потерпеть).
## Пример синтаксиса
```
# тест.rupy
если 1 == 1:
    печать(строка("Привет, мир!"))
иначе:
    печать(целое(87))

функ сложить_два_числа(а, б):
    вернуть а + б

печать(сложить_два_числа(1, 2)) # => 3
```

Если использовать старые замедленные слова из Питона, РуПитон кинет СинтаксическуюОшибку, ваши данные с ПК будут отправлены в соответствующие органы, а на вас заведено уголовное дело в соответствии со статьей № 168-ФЗ РФ.
## Использование
Для того, чтобы использовать интерпритатор РуПитона, нужно создать файл с расширением .rupy (допустим, тест.rupy).
Если вы на нашем отечественном дистрибутиве М-ОС, вам нужно создать "ссылку" на Питон. Делается это с помощью ``` alias питон='python' ```.

Далее запускаете файл интерпретатор.py.
```
╭(пплеер7@мос)-[~]
╰─₽ питон интерпретатор.py тест.rupy
```
